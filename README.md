# ⚽ Retrô FC - Sistema de Encomendas de Camisas Vintage

## 📖 Visão Geral
Este projeto é a evolução do sistema T-Burguer, adaptado para um novo segmento de negócio: **Retrô FC**, uma loja virtual especializada em camisas de futebol clássicas. 
A identidade visual foi totalmente reformulada, substituindo o amarelo/dourado por uma paleta moderna de Preto, Cinza e Azul. Os textos, imagens e regras de negócio foram adaptados (ex: "Ponto da Carne" foi substituído por "Modelo e Corte da Camisa").
